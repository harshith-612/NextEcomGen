# NextEcomGen

An innovative e-commerce application built with **Swift** and **SwiftUI**, delivering a modern and seamless shopping experience on iOS.

## 📱 About

NextEcomGen is a fully-featured e-commerce application designed to provide users with an intuitive and responsive shopping interface. Built entirely with Swift and SwiftUI, the app leverages Apple's latest UI framework to deliver a native, performant experience.

## 🎯 Features

- **Product Browsing**: Explore a wide catalog of products with detailed information
- **Search & Filter**: Find products quickly with advanced search and filtering options
- **Shopping Cart**: Add, remove, and manage items in your cart
- **User Authentication**: Secure login and account management
- **Order Management**: View order history and track purchases
- **Responsive Design**: Optimized UI for various iPhone screen sizes
- **Modern UI/UX**: Built with SwiftUI for a clean and intuitive interface

## 🛠️ Tech Stack

- **Language**: Swift
- **UI Framework**: SwiftUI
- **Platform**: iOS
- **Architecture**: MVVM.

## 📋 Requirements

- iOS 14.0 or later
- Xcode 12.0 or later
- Swift 5.3 or later

## 🚀 Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/harshith-612/NextEcomGen.git
   ```

2. Navigate to the project directory:
   ```bash
   cd NextEcomGen
   ```

3. Open the project in Xcode:
   ```bash
   open NextEcomGen.xcodeproj
   ```

4. Build and run the project:
   - Select your target device or simulator
   - Press `Cmd + R` to build and run
